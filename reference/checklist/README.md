# 设计自查清单索引

本目录为 **design-selfcheck** 的知识库：按维度拆分的完整自查条目，供 Agent 在 **Phase 3** 通过 `read_file` 读取。

**Skill 根目录**：与本文件的路径关系为 `design-selfcheck/reference/checklist/`（`SKILL.md` 位于 `design-selfcheck/SKILL.md`）。

## 阅读顺序（建议）

1. `01-需求分析.md`
2. `02-框架与布局.md`
3. `03-交互与特殊情况.md`
4. `04-界面细节.md`
5. `05-交付与走查.md`
6. `06-B端补充.md`

## 最小必读（上下文紧张时）

仅做快速评审时，至少读：**01、02、03**；涉及表格、批量、权限、金额等业务时必读 **06**。

## 维护约定

- 改条目：只改对应维度的 `.md`，不要在 `SKILL.md` 里重复粘贴全文。
- P1/P2 调整：在条目旁标注；重大变更可在此 README 末尾增加「变更记录」。
